# Pachimari
Discord bot for retrieving the latest in Overwatch League and Contenders news.

## Features 
* Player Information 
* Team Information
* Twitch Updates
* YouTube Updates
* Twitter Updates
* Match updates
* News Updates
* Schedule Updates
* Live Match Statistics
* Predition polls
* Memes!
* Standings
* ADD CONTENDERS
